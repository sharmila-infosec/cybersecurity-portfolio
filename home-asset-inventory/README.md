# Home Asset Inventory

This activity focuses on creating an inventory of devices connected to a home office network. The goal is to identify network assets, describe their access, owner, location, and classify their sensitivity level.

## Activity Summary

The inventory includes three home network devices:

- Laptop
- Printer
- External hard drive

Each device was evaluated based on:
- Network access
- Owner
- Location
- Security notes
- Sensitivity classification

## Skills Practiced

- Asset management
- Asset inventory creation
- Risk evaluation
- Sensitivity classification
- Home network security awareness
