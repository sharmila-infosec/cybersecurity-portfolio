# File Permissions in Linux

## Project Description

In this project, I examined file and directory permissions in Linux to identify unauthorized access and update permissions using Linux commands. I reviewed existing permissions, interpreted permission strings, and modified file and directory access using the `chmod` command to follow organizational security requirements.

## Skills Demonstrated

- Linux file permissions
- File ownership
- Directory permissions
- `ls -la`
- `chmod`
- Hidden files
- Access control
- Authorization management
- Linux system administration

## Files

- File permissions in Linux.pdf

## Summary

This project demonstrates how Linux file permissions help protect sensitive files and directories. By reviewing and updating permissions, I ensured that only authorized users had the appropriate level of access, reducing security risks and maintaining system integrity.
